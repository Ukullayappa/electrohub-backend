# HubElectro Backend API

Backend service for the HubElectro Full Stack E-Commerce Platform. Built using Node.js, Express.js, PostgreSQL, Passport.js, and REST APIs. Responsible for authentication, authorization, product management, cart operations, and order processing.

## Related Repositories

Frontend:
https://github.com/Ukullayappa/electrohub-frontend

Backend:
https://github.com/Ukullayappa/electrohub-backend

## Features

* RESTful API Architecture
* User Authentication
* Session-Based Authentication
* Passport.js Integration
* Role-Based Access Control (RBAC)
* Product Management APIs
* Shopping Cart APIs
* Order Processing APIs
* PostgreSQL Database Integration
* Secure Environment Configuration

## Installation

### Clone Repository

```bash
git clone https://github.com/Ukullayappa/electrohub-backend.git

cd electrohub-backend
```

### Install Dependencies

```bash
npm install
```

### Start Server

```bash
npm start
```

### Start Server

```bash
npm start
```
